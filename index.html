What is CUFA?

CUFA or CIT-University Facebook Analyzer is a web application that could retrieve data from the official Cebu Institute of Technology University Facebook pages and analyze the data of the specific Facebook pages. The web application features a dashboard that will display the key performance indicators from the Facebook page that is used by the Facebook page admin to show and determine the Facebook pageâ€™s performance and how individuals interact with their pages in general.

Page Performance

Welcome to CUFA's page performance section, this page show the analytics of page performance of a selected page. At the upper right part are the filters which lets you filter what page, specific date periods, and what month, below are KPI's and graph that shows analytics on likes, reach, total views, and other page analytics related to page performance scrolling down shows indicators an arrow up with a danger or red color which shows a decline or decreased in performance and a success or green color whichs shows an increased.

Page Comparison

Welcome to CUFA's page comparison section, this page gives the user a complete picture of how the performance of each Facebook page compares. The user can also select only specific pages to compare, which can be two or more Facebook pages. Post and Response analysis is also shown in this page that displays the number of Posts in a week, Weekly Response Rate, Post per day and Average Response Time. Finally, the user can use the built-in Power BI function to export data from a specific visualization in Excel Format.

Drill Down

Drill Down Functionality

  • Drill Down is a feature in powerBI graphs by default
  • Drill Down is represented by two consecutive down arrows
  • By default the graph displays data by week
  • By clicking drill down button it will display data by day
  • By clicking the drill up button represent and up arrow the graph will essentially return to default display which is data by week
  • Clicking the drill up button again will display data by month
  • Not to be confused by turn on drill represented by only a one arrow
  • Which has another function and that is to enable users to drill down specific locations in the graph
  • To determine that this functions is in used a black circle background appears behind it and if you hover through it a message shows that the drill is turned on
